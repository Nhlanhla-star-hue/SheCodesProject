# 🧵 Sewing Landing Page

A responsive landing page created as part of the **SheCodes Basics Add-on** workshop. This project was designed to reinforce my understanding of HTML, CSS, and JavaScript by building a complete, interactive webpage from scratch. The SheCodes Basics Add-on workshop focuses on expanding front-end development skills through hands-on projects and practical application of HTML, CSS, JavaScript, and responsive design concepts.

🌐 **Live Demo:** https://nhlanhla-star-hue.github.io/SheCodesProject/

## 📖 Project Overview

This project showcases my growing front-end development skills through the creation of a responsive and interactive landing page based on one of my personal interests—sewing and fashion design.

The webpage combines modern HTML, CSS, and JavaScript techniques to create an engaging user experience while demonstrating responsive layouts, animations, interactive image galleries, and theme switching functionality.

* 🌙 Dark Mode / Light Mode toggle
